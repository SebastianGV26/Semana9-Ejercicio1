Ejercicio 2

Definicion y ejemplo de freezear un objeto

Es la forma de congelar un objeto para no poder realizar absolutamente nada, ni modificar, agregar o eliminar una propiedad

Object.freeze(objeto);

Ejercicio 3

Definicion y ejemplo de sellar un objeto

Es la forma de bloquear un objeto para poder modificar, pero no poder agregar o eliminar una propiedad

Object.seal(objeto);

Definicion y ejemplo de unir o sumar dos objetos

Se suman los objetos para poder extraer información precisa de cada uno de ellos, según sea la necesidad

const infoPersona = Object.assign(objeto1,objeto2);
